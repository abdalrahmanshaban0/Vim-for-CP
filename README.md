To Build and Run ---> Ctrl + F9 <br>
To Paste test cases in input.txt while in main ---> Ctrl + t <br>
To Close any splited tab ---> Ctrl + q <br>
To Split a file from NERDTree ---> s <br>
Auto Indent from what you stand to down ---> =G
